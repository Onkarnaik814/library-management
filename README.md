# Library Management System (Python)

This project is a simple command-line based **Library Management System** built in Python. It allows users to select an option, choose a book, and automatically open related learning resources in their default web browser.

---

## üìå Features

* Displays a welcome message and prompts the user for input.
* Allows user to exit or proceed to the book menu.
* Provides a list of programming-related books:

  * C
  * Java
  * HTML
  * Python
* Opens a corresponding online resource for the selected book using the `webbrowser` module.
